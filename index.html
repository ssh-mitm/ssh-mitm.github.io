SSH-MITM – ssh audits made simple

A man-in-the-middle SSH server for security audits with publickey authentication, session hijacking and file manipulation.

Get Started

Key Features

  • Publickey authentication & FIDO token phishing
  • Hijacking and logging of SSH sessions
  • SCP/SFTP file manipulation
  • Dynamic port forwarding with SOCKS 4/5
  • MOSH interception
  • Client vulnerability audits
  • Plugin support

Quickstart

Install SSH-MITM with pip:

python3 -m pip install ssh-mitm

Start the MITM server and connect via SSH:

ssh-mitm server --remote-host 192.168.0.x
ssh -p 10022 user@proxyserver

That’s it – credentials and session logs are captured.
For detailed installation methods and advanced usage, see the documentation.
